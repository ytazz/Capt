NAME : 	
N-Step Capturability Graphical User Interface 

VERSION : 	
1.0

RELEASE DATE:
October 2010

REQUIREMENTS:
MathWorks Matlab, tested with Matlab versions 7.4/2007a. Should work
with this version and higher.

EXECUTE:
Run NStep.m from within Matlab

This N-Step Capturability GUI is supplied as an addition to the paper:
"Capturability-Based Analysis and Control of Legged Locomotion, 
Part 1: Theory and Application to Three Simple Gait Models."
